# Go Interview Prep

Compact interview-prep workspace for a C# developer ramping into Go/Goland.

## Contents

- `CLAUDE.md` - behavioral coding guidance copied from `multica-ai/andrej-karpathy-skills`.
- `GO_FOR_CSHARP_CHEATSHEET.md` - short Go cheat sheet focused on syntax, concurrency, initialization, and common landmines.
- `skills/cc-skills-golang` - full copied contents of `samber/cc-skills-golang`.
- `skills/visualise` - full copied contents of `bentossell/visualise`.
